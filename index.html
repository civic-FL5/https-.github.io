Hondaのcivic type Rについて

私は小さいころから車が好きで自分の好きな車についてまとめました。

Honda civic type RはHonda車の中で上位に入るスポーツ性能を持っている車です。

つい先日9月2日最後の純ガソリンエンジン車になるであろう新型のcivic type Rが発売されました。


↑今回出た新型　FL5

スペック詳細

  • 馬力330馬力

  • 最大トルク420N・m

  • 2.0L VTEC TURBO

  • 6速マニュアルトランスミッション

  • WLTCモード燃費　12.5km/L

  • 駆動方式FF（前輪駆動）

実はcivic type Rは今までものすごい記録を残している


  • 先代モデルのFK8ともう一つ前のFK2はドイツの高低差がものすごくあるニュルブルクリンク北コースで量産モデルのFF最速を記録している


    • ↑先代モデルFK8（自分ははじめてこの車見たとき一目ぼれした）


    ↑さらに1つ前のモデルFK2

    今現在新型モデルもニュルブルクリンクにてタイムアタック中↓

    https://www.youtube.com/watch?v=vScAXoyoov8&t=3s

  • ライバルメーカーは後部座席なしやエアコン、ナビレスの中新記録を立てたがcivicはすべての装備がついている状態で記録を樹立した

  • 一応新型のFL5は日本の鈴鹿サーキットでFF量産車最速を記録している（実はこの記録先代FK8の限定車が叩き出した記録「エアコン、ナビレスの限定車」を上回った）

    • 今回の新型のすごいところ

  • 価格がものすごく安い、お買い得、今回の新型モデルで500万円を超えるかと予想していたがなんと税込みで499万円で買うことができる（何も装備はないが）

  • エンジン出力を3段階選ぶことができ、シーンによって選ぶことができる（＋Ｒ、SPORTS、COMFORT）

  • チャイルドシートもつけれたりする

  • 先代FK8よりも10馬力upトルクは20N・m upしている

自分はこの車が好きなので生産終了する前に購入したいと思います。（今月中にはオーダー多分入れます...）
